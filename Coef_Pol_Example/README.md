\# SQ Polynomial Coefficients Example



Esta carpeta contiene archivos JSON de ejemplo con coeficientes asociados a la construcción de curvas de variación solar regular dentro de SIAPV-API.



Estos archivos permiten documentar el formato de salida esperado después del procesamiento de días quietos.



\## Contenido



\- Archivos `.json` con coeficientes polinomiales de ejemplo.

\- Resultados derivados del ajuste de la variación regular.



\## Nota



Los archivos incluidos son ejemplos reducidos y no corresponden a la totalidad de los productos operativos generados por el sistema.
